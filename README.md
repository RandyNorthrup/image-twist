# Image Twist

Image Twist is a local browser app for applying visual twist effects to an image and exporting the result as a PNG.

## Twist Library

The twist library contains +100 unique effects, including:

- `Cyanotype`
- `Emboss`
- `Water Ripple`
- `Ordered Dither`
- `CRT Mask`
- `Neon Edge`
- `Kaleidoscope`
- `Glitch`
- `Poster Punch`
- `Vignette Frame`

## Setup

No install or build step is required.

1. Download or clone this folder.
2. Open `index.html` in a modern browser.
3. Enjoy!

## How To Use

1. Drop an image onto `Drop image`, or choose an image file.
2. Pick an effect from `Choose twist`, or press `Twist` for a random effect.
3. Adjust the Intensity slider to adjust effect level.
4. Press `Auto Party` to cycle through random effects.
5. Press `Download PNG` to save the current image.
